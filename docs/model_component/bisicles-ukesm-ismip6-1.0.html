BISICLES-UKESM-ISMIP6-1.0 — UniCiCles (Unified Model-CISM-BISICLES) is a package combining BISICLES with an interface that obtains boundary conditions from Unified Model or JULES data.

Family: BISICLES uses adaptive-mesh Chombo libraries for ice sheet modeling with SSA- momentum approximation.
