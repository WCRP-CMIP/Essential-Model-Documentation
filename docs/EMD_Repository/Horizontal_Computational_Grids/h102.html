What is this? This is a horizontal computational grid using a Arakawa C-grid staggering arrangement. It contains 1 subgrid.

Staggering Arrangement

Arakawa C-grid
The Arakawa C grid places velocity-related quantities at the centres of mass cell edges, such that each component is perpendicular to its edge.
